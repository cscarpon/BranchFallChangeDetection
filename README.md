# BranchFallChangeDetection

Python code for detecting changes in branch fall structures using voxelization and 3D point cloud processing.

A data sample is given in the data folder, it is composed of the same tree scanned at two epochs 2022 and 2023.

## Setup
- Python 3.10+
- Open3D

## Usage
- Place your data in the `data/` directory.
- Run `python change_detection.py`.

## Credits and citation

This package originates from the following study:

Karl Montalban et al., Exploring the impact of ice storm on urban forests and branch fall using mobile LiDAR, Urban Forestry & Urban Greening, 2025.
